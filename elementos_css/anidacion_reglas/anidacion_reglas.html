Ejemplo de Anidación de Reglas

La anidación de reglas se usa para aplicar estilos a un elemento dentro del contexto de otro elemento. Además, reduce la necesidad de repetir selectores para elementos relacionados.

Sin anidar, hay que declarar cada selector explícitamente y por separado:
